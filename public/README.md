Place website assets in this `public` folder.

Recommended structure:

- `public/images/` for JPG, PNG, WebP, and SVG images
- `public/icons/` for small UI icons and logos
- `public/illustrations/` for larger artwork and hero graphics
- `public/fonts/` for self-hosted font files

Usage from HTML/CSS:

- HTML: `src="public/images/hero-card.png"`
- CSS: `background-image: url("public/illustrations/hero-orbit.svg");`

If you add assets here, I can wire them into the page for you.
