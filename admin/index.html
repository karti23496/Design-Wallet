Design Wallet Admin

Manage listings in Google Sheets.

This website now reads the catalog directly from your Google Sheet instead of the old Git-based CMS. Add or edit rows in the sheet and refresh the site to see the updated listings.

Open Google Sheet Open Website
  • Required columns can include: Image, Title, Subtitle, Categories, Pricing, Link, Thumbnails, and Slug.
  • Categories should be comma-separated if you want one listing to appear in multiple filters.
  • Pricing values like FREE, FREEMIUM, and PAID are supported automatically.
