# Designer Wallet Website

This is a basic starter pack for the Designer Wallet website, including HTML, CSS, and JavaScript files. Now includes Netlify CMS for content management.

## Files Included

- `index.html`: The main HTML structure of the website.
- `style.css`: Basic CSS styles for layout and appearance.
- `script.js`: Simple JavaScript for interactivity.
- `config.yml`: Configuration for Netlify CMS.
- `admin/index.html`: Admin interface for Netlify CMS.

## How to View the Website

1. Open `index.html` in your web browser.
2. Alternatively, serve the files using a local web server for better functionality.

## Using the CMS

1. Deploy the site to Netlify (connect your Git repository).
2. Access the admin panel at `yoursite.netlify.app/admin`.
3. Log in with your Netlify credentials to edit content.

Note: Netlify CMS works best with static site generators. For full functionality, consider migrating to a framework like Next.js.

## Usage

- Customize the content in `index.html` to fit your needs.
- Modify `style.css` to change the appearance.
- Add more JavaScript functionality in `script.js`.

## Troubleshooting

- If the styles don't load, ensure `style.css` is in the same directory as `index.html`.
- If JavaScript doesn't work, check the browser console for errors.
- For CMS issues, ensure the site is deployed on Netlify.
